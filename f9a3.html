[FBCTF2019]RCEService

  • 原理参考 p 🐮 的文章即可

https://www.leavesongs.com/PENETRATION/use-pcre-backtrack-limit-to-bypass-restrict.html

也就是说,php 的PCRE回溯次数是有限制的, var_dump(ini_get('pcre.backtrack_limit)) 可以看到这个限制是1000000次,当超过最大限度的时候将返回false 而不是0或者1 ,就可以用于绕过waf

如何来防御呢? 使用preg_match() 对字符串进行匹配的时候,一定使用===全等号来判断即可,即使正则执行失败,也不会返回false

非预期:

preg_match 函数只能匹配第一行数据,用%0a 换行符即可绕过,在用JavaScript伪协议导致xss的时候也可以用这个点

[强网杯 2019]Upload

  • 通过dirsearch得到源码泄露(www.tar.gz)
  • php 反序列化实现任意文件名修改 getshell
  • 序列化时会将命名空间进行序列化,因此在存在命名空间的反序列化攻击利用的时候要添加命名空间

分析:

将源码解压后拖进PHPstorm, 首先在index.php和Register.php 中,存在断点,是hint了,

分别是:

和:

很明显了是反序列化漏洞,将一些操作将原本不可操作的值进行覆盖,而达到类似可控的效果

反序列化点在index.php中的login_check方法中,将cookie取出,然后进行数据上的格式化操作, 因此 cookie 作为本题的入口

在Profile.php 中,$this->except 在反序列化时可控,就可以出发__call调用任意类方法

在Register.php 中,存在__destruct方法, 其中$this->register, $this->checker 在反序列化时也控(理解这步很关键…本菜鸡感觉有点类似变量覆盖的味道)

如果我们将 $this->checker 赋值为 Register 类，而 Register 类没有 index 方法，所以调用的时候就会触发 __call 方法,进而修改Profile中的参赛, 这样就形成了一条完整的攻击链。

[CISCN 2019 初赛]Love Math

非常有意思的题, 参考 Smi1e 师傅的文章来做的…. tql 啊

原文链接

题目禁用了空格, 制表符,换行符,会出,单双引号,反引号,以及[]

并做出了长度限制,不能超过80. 可以使用的是一些数学函数, 其中我们要利用的一个函数就是base_convert() 函数,该函数可以返回任意字母,但是无法返回_ * D等特殊字符.

最后RCE 的点为: eval('echo '.$content.';'); 单引号包裹,并且有了分号,我们只需要构造$content 为我们想要执行的函数即可

思路: 通过base_convert()函数构造出函数名,然后用()来调用执行

现在我们需要想办法读取flag.php的内容，三条路：
1、使用php函数readfile等函数读取文件，但是需要flag.php中的.。
2、使用system等命令执行函数配合通配符*读取文件，但是需要*。
3、使用$_GET全局变量手动传入参数getshell。 上面的三种方法都建立在字符长度小于80的条件下。

base_convert(1751504350,10,36)(base_convert(784,10,36)) 执行ls

得到两个index.php

方法一: 利用hexdec 将ASCII转为10进制, dechex() 将十进制转换为16进制,再利用hex2bin() 将16进制换为任意ASCII字符

. 在ASCII 码表中 10进制为46 ,16进制为2E

为了缩短字符长度,可以将函数base_convert赋值给一个短变量名,由于白名单限制,传给最短的pi

cat *的16进制, 为636174202a

但是只有index.php , 所以我们要getshell,去读 根目录下的 flag /flag

方法三: getshell

$pi=base_convert;$pi=$pi(53179,10,36)^$pi(1109136,10,36);${$pi}{0}(${$pi}{1}) 77 个字符

总结:

  • 难点: 函数的掌握和灵活应用
  • fuzz
  • 反复看别人的wp, 学习大佬的思路,而不是简单的为了做题而做题

fuzz 异或代码:

intruder 设置为 Cluster bomb 模式

payload1 位 0-9的数字, payload2 为字母, 设置Grep - Match 为_ G E T按着找我们需要的符号即可, 发现1^n=_ 5^a=T 7^p=G 0^d=E 然后 $ 没有被ban,我们就可以构造超全局变量$_GET 来getshell了~

[CISCN2019 总决赛 Day2 Web1]Easyweb

  • 扫目录robots.txt => *.php.bak => image.php.bak
  • 文件上传, 短标签绕过对PHP的限制
  • 单引号逃逸盲注

可以看到这里把 id 或者 path 中的\\0 与 %00, \\' 替换为了空, 那么我们可以用 BJD 那个简单注入的思路来进行盲注, => 即利用addslashes函数转义多添加的一个\ 来注入掉id后面的' ,利用# 来注释掉后面的查询语句来进行盲注

来复习一下,MYSQL 中的 IF语句 :

IF(expr,v1,v2)

其中：表达式 expr 得到不同的结果，当 expr 为真是返回 v1 的值，否则返回 v2.

登陆进去之后, 是一个文件上传的功能,那么这么明显了 肯定得搞到shell 才能拿flag.. 可惜我挖了这么几次洞还没遇见文件上传漏洞=.=

这里我随便传了个图片马, 改为PHP后缀,✖. 访问上传的日志文件, 该文件为.php 文件, 并且将我们上传的文件名输出到了页面,那么我们就在filename里面写一句话试试, 发现任然不能上传,在之前的无字母getshell中, 其实就遇到过,php 的另外一种格式

发现这样蚁剑还是连不上, 于是用下面那种, <?= @eval($_POST[1]);?> 即可连上, 在虚拟终端里面cat /flag 即可
